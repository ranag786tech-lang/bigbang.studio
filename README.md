# Big Bang Studio 🚀

**One Prompt. Infinite Creation.**

Meet DigiD — Your All-Rounder AI for Code, Images, Apps, Stories & More.

Build production-ready websites, apps, and digital experiences by just describing them in plain language. No coding required.

## 🌟 Live Demo

- **Landing Page**: https://bigbang-studio.vercel.app/
- **Playground**: https://bigbang-studio.vercel.app/playground.html

## ✨ Features

- **DigiD AI**: Chat-based code generation. Describe what you want, get working HTML/CSS/JS
- **Live Preview**: See your website update in real-time as DigiD codes
- **Files Tree**: Browse and view generated code files instantly
- **Explain & Debug**: DigiD explains code line-by-line in simple language
- **DigiD Memory**: Remembers your last 5 prompts across sessions
- **One-Click Deploy**: Deploy to Vercel or share link instantly
- **Glassmorphism UI**: Cosmic space theme with neon cyan/pink aesthetics

## 🛠️ Tech Stack

- **Frontend**: Pure HTML5, CSS3, Vanilla JavaScript
- **Styling**: Tailwind CSS via CDN + Custom Glassmorphism
- **Fonts**: Orbitron, Space Grotesk via Google Fonts
- **Animation**: Canvas Starfield + CSS Transitions
- **Storage**: LocalStorage for projects & memory
- **Deploy**: Vercel
